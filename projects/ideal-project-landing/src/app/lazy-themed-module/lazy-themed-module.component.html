This component was lazy loaded. And it's themed - just click on the button in the header

All Angular components should be themed, but for demonstration purposes, I only themed this component.

I think all page modules should be loaded lazy.

In Angular, we have a special mechanism for lazy loading a module - usually, to lazy load a function module, we use a router, define routes, and use "loadChildren" to dynamically import the function module.

In this case module was loaded lazy with dynamic import without routing.
